## Clonar y Ejecutar Localmente

Clonar el proyecto

```bash
  git clone <repository url>
```

Ir al directorio del proyecto

```bash
  cd my-project
```

Instalar dependencias

```bash
  npm install
```

Correr el servidor Vite

```bash
  npm run dev
```
